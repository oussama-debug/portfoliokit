A modern meeting scheduling platform built with Next.js and Supabase.

## Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: Supabase
- **Monorepo**: Turborepo + pnpm
- **Forms**: React Hook Form + Zod
- **State**: Zustand

## Getting Started

```bash
pnpm install
pnpm dev
```
